# AETHER - Luxury Immersive Technology Studio

A premium, award-worthy landing page for AETHER, a fictional luxury immersive technology studio that crafts transcendent digital experiences blending physical and virtual worlds.

![AETHER Preview](assets/images/hero-bg.jpg)

## Project Overview

AETHER is a fully responsive, production-ready landing page designed to showcase a luxury immersive technology studio. The website features cutting-edge web design techniques including glassmorphism, smooth scroll animations, custom cursor, particle effects, and a sophisticated dark/light theme system.

## Business Concept

**AETHER** is a luxury immersive technology studio founded in 2016. We specialize in creating bespoke digital experiences for luxury brands, cultural institutions, and visionary clients. Our services include:

- **Immersive Installations** - Multi-sensory environments that transport audiences
- **Spatial Computing** - AR, VR, and mixed reality experiences
- **Generative Art** - Algorithmic masterpieces powered by custom code and AI
- **Projection Mapping** - Transforming any surface into a stunning canvas
- **Holographic Displays** - Three-dimensional visual experiences floating in mid-air
- **Interactive Experiences** - Gesture-controlled, touch-responsive installations

### Target Audience
- Luxury brands (fashion, automotive, hospitality)
- Museums and cultural institutions
- Art galleries and exhibition spaces
- Premium retail and flagship stores
- Event organizers and brand activations

## Installation

1. Clone or download this repository:
```bash
git clone https://github.com/oltionshumollii/aether-luxury-immersive-studio.git
```

2. Navigate to the project directory:
```bash
cd aether-luxury-immersive-studio
```

3. Open `index.html` in your browser, or serve with a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .

# Using PHP
php -S localhost:8000
```

## Usage

### Deployment
This project is ready for deployment on:
- **GitHub Pages** - Push to a GitHub repository and enable Pages
- **Netlify** - Drag and drop the folder or connect your repo
- **Vercel** - Import your repository
- **Any static hosting** - Upload all files to your server

### Customization

#### Colors
Edit CSS variables in `css/style.css`:
```css
:root {
    --color-gold: #D4AF37;
    --color-bg: #0A0A0F;
    /* ... */
}
```

#### Content
- Update text content directly in `index.html`
- Replace images in `assets/images/`
- Modify team member details in the Team section
- Update portfolio items with your own projects
